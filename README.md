# Pluto Marketing Website

The marketing website for Pluto - the git workflow tool for AI-first teams.

## About Pluto

Pluto transforms how AI-first teams collaborate on code by providing a workflow that captures every change as "fibers", organizes them into meaningful "threads", and weaves everything together for seamless PR integration.
